# dirichlet-problem-local-martingales

Python code for demonstrating the connections between
the Dirichlet problem and continuous local martingales.

## Files

- The source code is [dplm.py](./dplm.py).
- The plots can be produced by running [script.py](./script.py).

## Usage

```
python3 script.py
```

## Output

The plots are exported to [graphics/](./graphics)
